# ✨ OTTO - Features Guide

## 🎤 Voice & Chat Features

### 1. Two-Way Conversation
- **Chat Mode**: Type your messages
- **Voice Mode**: Speak naturally
- **Dictation Mode**: Click mic to record
- **3D Orb Animation**: Visual feedback while speaking

### 2. Wake Word Detection
- **Customizable**: Set any word ("Otto", "Hey", etc)
- **Always Listening**: Background listening mode (saves battery)
- **Manual Activation**: Click button to activate
- **Keyboard Shortcut** (PC): Ctrl+A (coming soon)

### 3. Voice Customization
- **Choose Voice**: Multiple voices available
- **Accent Selection**: Different accents (US, UK, BR, etc)
- **Speed Control**: Adjust speaking speed
- **Tone**: Professional, casual, friendly, etc
- **Voice Test**: Listen before choosing

### 4. Multi-Language Support
- Portuguese, English, Spanish, French, German, Italian, Japanese, Chinese
- Switch languages anytime
- Context maintained across languages
